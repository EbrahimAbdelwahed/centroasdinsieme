# Sistema di memoria del progetto

Questa cartella contiene note operative per gli agenti che lavorano sul progetto.

## Cartelle

- `piani/`: piani di implementazione, decisioni operative, cronologia e log di lavoro.
- `handoff/`: note finali dopo una fase di implementazione, utili agli agenti successivi.
- `memorie/`: informazioni stabili e importanti del progetto, da consultare prima di lavorare.

## Convenzioni

- Usare file Markdown.
- Preferire nomi descrittivi con data ISO quando il contenuto e' legato a una fase di lavoro.
- Non duplicare note temporanee dentro `memorie/`: le memorie devono contenere vincoli e decisioni ancora valide.
- Dopo ogni implementazione sostanziale, lasciare un handoff con cosa e' stato fatto, cosa resta aperto e quali file sono stati toccati.
